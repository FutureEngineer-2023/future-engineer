### 優點：

1.高效能：一些現代馬達設計能夠提供高效能和效率，將電能轉換為機械能，減少能源浪費。

2.高扭矩：某些馬達能夠提供高扭矩輸出，使其適用於需要大力矩的應用。

3.精確控制：許多馬達具有精確的控制能力，可以實現精確的運動控制和定位。

4.長壽命：良好設計和製造質量的馬達通常具有較長的使用壽命，降低維修和更換成本。
### 缺點：

1.成本：高性能馬達通常具有較高的成本，特別是在特殊應用或特定要求的情況下。

2.體積和重量：某些馬達可能相對較大且重量較重，這可能在空間有限或重量敏感的應用中造成問題。

## 轉向控制結構 
**我們列出了兩種方案型比較。**
### 1.伺服馬達搭配連桿轉向

![image](https://github.com/yy0106yy/future-engineer/blob/main/%E7%A7%BB%E5%8B%95%E6%80%A7%E7%AE%A1%E7%90%86Mobility%20Managemen/%E8%BD%89%E5%90%91%E7%B5%90%E6%A7%8B/%E9%80%A3%E6%A1%BF%E8%BD%89%E5%90%91%E7%B5%90%E6%A7%8B%E5%9C%96.png)
![image](https://github.com/yy0106yy/future-engineer/blob/main/%E7%A7%BB%E5%8B%95%E6%80%A7%E7%AE%A1%E7%90%86Mobility%20Managemen/SG-90.png)
### 優點：

1.小巧輕便：伺服馬達的體積小且重量輕，適合空間有限的應用場合。

2.高精度：伺服馬達具有相對較高的精確度和位置控制能力，適用於一些需要精確定位的應用。

3.簡單控制：伺服馬達通常只需要使用脈衝寬度調製（PWM）信號來控制，相對簡單易用。

4.相比於齒條轉向設計簡單，且誤差能減少。

### 缺點：

1.功率限制：伺服馬達的功率輸出相對較低，適用於較小負載或低功率要求的應用。

2.轉速限制：伺服馬達的轉速通常較低，不適合需要高速運動的應用。

3.耐久性限制：由於其經濟實惠的設計，伺服馬達可能在長時間或高負載運行下的耐久性較差。

## 馬達驅動板

### 1.TB6612FNG

![image](https://github.com/yy0106yy/future-engineer/blob/main/%E7%A7%BB%E5%8B%95%E6%80%A7%E7%AE%A1%E7%90%86Mobility%20Managemen/MMD-003286.jpg)

![image](https://github.com/yy0106yy/future-engineer/blob/main/%E7%A7%BB%E5%8B%95%E6%80%A7%E7%AE%A1%E7%90%86Mobility%20Managemen/FCNO9VGJHATTMWR%20(1).jpg)
以下是TB6612FNG馬達控制板的一些基本規格和特點：

1.輸入電壓：5V-13.5V

2.馬達驅動電流：最大1.2A（平均）/ 最大3.2A（峰值）

3.適用馬達類型：直流馬達

4.控制接口：PWM控制信號（用於速度調節）、DIR控制信號（用於轉向控制）

5.馬達輸出通道：2個（可以同時控制兩個馬達）

6.過電流保護：內置過電流保護功能

7.熱關閉保護：內置熱關閉保護功能

8.內部反向極性保護二極體：內置反向極性保護二極體，以防止馬達回饋電壓對芯片造成損害。

9.尺寸：通常為20mm x 11mm x 3mm

### 優點：

輸入電壓範圍寬，適用於不同電源需求。

馬達驅動電流較高，能夠驅動較大功率的馬達。

具備過電流保護和熱關閉保護功能，提供額外的安全性。

尺寸較小，適合應用於有空間限制的項目。

### 缺點：

控制接口相對較少，可能不夠彈性。

需要額外的電阻和電容來設置PWM頻率和過電流保護閾值。

需要外部反向極性保護二極體，以保護芯片免受馬達回饋電壓的損害。

### DRV8833 
![image](https://github.com/yy0106yy/future-engineer/blob/main/%E7%A7%BB%E5%8B%95%E6%80%A7%E7%AE%A1%E7%90%86Mobility%20Managemen/XRB8z.jpg)

1.輸入電壓：2.7V-10.8V

2.馬達驅動電流：最大1.2A（平均）/ 最大2.5A（峰值）

3.適用馬達類型：直流馬達

4.控制接口：PWM控制信號（用於速度調節）、DIR控制信號（用於轉向控制）

5.馬達輸出通道：2個（可以同時控制兩個馬達）

6.過電流保護：內置過電流保護功能

7.低電壓保護：內置低電壓保護功能

8.內部反向極性保護二極體：內置反向極性保護二極體，以防止馬達回饋電壓對芯片造成損害。

9.尺寸：通常為20mm x 11mm x 3mm

### 優點：

電源電壓範圍適中，適用於一般馬達應用。

具備過電流保護和低電壓保護功能，提供額外的安全性。

控制接口簡單，易於使用和集成到專案中。

尺寸較小，適合應用於有空間限制的項目。

### 缺點：

馬達驅動電流較低，不能驅動過大功率的馬達。

控制能力有限，無法提供高級的馬達控制功能（如步進馬達控制）。

需要外部反向極性保護二極體，以保護芯片免受馬達回饋電壓的損害。

### L298N
![image](https://github.com/yy0106yy/future-engineer/blob/main/%E7%A7%BB%E5%8B%95%E6%80%A7%E7%AE%A1%E7%90%86Mobility%20Managemen/L298N-Motor-Driver-pic2.jpg)

1.輸入電壓：7V-35V

2.馬達驅動電流：最大2A（平均）/ 最大3A（峰值）

3.控制接口：4個輸入（IN1、IN2、IN3、IN4）和2個使能輸入（ENA、ENB）

4.馬達輸出通道：2個（可以同時控制兩個馬達）

5.控制信號類型：使用PWM信號控制速度，使用DIR信號控制轉向

6.過電流保護：內置過電流保護功能

7.過熱保護：內置過熱保護功能

8.尺寸：通常為43mm x 43mm x 27mm
### 優點：

輸入電壓範圍寬，適用於不同電源需求。

能夠提供較高的馬達驅動電流，能夠驅動較大功率的馬達。

控制接口多，具備較高的彈性和控制能力。

可以控制直流馬達和步進馬達，提供更廣泛的應用。

### 缺點：

需要外部電阻和電容來設置PWM頻率和過電流保護閾值。

尺寸較大，需要更多的空間來安裝和集成。

由於較舊的設計，效率較低，可能產生較多的熱量。

### 結論: 

由上文分析，我們選用TB6612FNG，

**原因:**

電源電壓範圍：TB6612FNG的電源電壓範圍為5V-13.5V，能夠滿足2顆4.2V電池的供電需求。

馬達驅動電流：TB6612FNG的最大平均馬達驅動電流為1.2A，最大峰值電流為3.2A。根據N20馬達的規格，TB6612FNG可以很好地驅動GA12-N20，並提供足夠的電流。

控制接口：TB6612FNG具有PWM控制信號和DIR控制信號的接口，方便控制馬達的速度和轉向。

尺寸和易用性：TB6612FNG的尺寸相對較小，易於安裝。

## 底盤設計 

起初我們試著採用**遙控車的底盤**進行改裝，雖然方便省時，但是遙控車的轉向不夠達到我們的要求，也不容易改裝。
接著我們改用**樂高積木**直接組裝出一台，跟遙控車相比，樂高自由調整度更高，能夠設計出更好的轉向，還可以用內建的馬達進行控制，只可惜組裝完成的體積跟重量太大，導致速度跟敏捷無法有好的效果。
最後我們只好選擇靠**3D列印**自己建模。這樣的好處就是尺寸、版面配置都可以自己設定。
